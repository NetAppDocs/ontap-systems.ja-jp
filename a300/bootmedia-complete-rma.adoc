= 障害のある部品を NetApp-AFF A300 に戻します
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ブート メディアを交換した後、故障した部品をNetAppに返却してください。AFF A300システムは、手動のブート メディア リカバリ手順のみをサポートします。自動ブート メディア リカバリはサポートされていません。

障害が発生したパーツは、キットに付属のRMA指示書に従ってNetAppに返却してください。 https://mysupport.netapp.com/site/info/rma["パーツの返品と交換"]詳細については、ページを参照してください。
