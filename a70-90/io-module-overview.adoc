= I/Oモジュールの追加と交換- AFF A70-90Kの概要
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ストレージシステム内の障害が発生したI/Oモジュールは、同じタイプのI/Oモジュールに交換することも、別の種類のI/Oモジュールに交換することもできます。空きスロットのあるシステムにI/Oモジュールを追加することもできます。
