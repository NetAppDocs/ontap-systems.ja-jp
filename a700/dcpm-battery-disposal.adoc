= バッテリの廃棄- AFF A700
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
バッテリのリサイクルまたは廃棄に関する地域の規制に従って、バッテリを廃棄する必要があります。バッテリを適切に廃棄できない場合は、キットに付属する RMA 指示書に従って、バッテリをネットアップに返却する必要があります。
