= ASA A70の主な仕様
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
以下は、単一の高可用性ペア内のASA A70 ストレージ システムの選択仕様です。このストレージ システムの完全な仕様については、 NetApp Hardware Universe (HWU) をご覧ください。



== 主な容量、メモリ、フォーム ファクタ、およびONTAPバージョンの仕様

* プラットフォーム構成: ASA A70 シングルシャーシ HA ペア
* 最大生容量: 2.6622 PB
* メモリ: 256.0000 GB
* フォームファクター: 2 つの HA コントローラと 48 個のドライブ スロットを備えた 4U シャーシ
* ONTAPバージョン: ONTAP: 9.16.1P2
* PCIe拡張スロット: 18
* 最小ONTAPバージョン: ONTAP 9.16.0




== スケールアウトの最大値

* タイプ: NAS
* タイプ: SAN; HA ペア: 6; 物理容量: 16.0 PB / 14.2 PiB; 最大メモリ: 1536 GB
* タイプ: HA ペア; 物理容量: 2.7 PB / 2.4 PiB; 最大メモリ: 256.0000

* プロトコル: USB 600 Mbps; ポート: 2




== ストレージネットワークをサポート

* FC
* iSCSI
* NVMe/FC
* NVMe/TCP




== システム環境仕様

* 標準電力: 6938 BTU/時
* 最悪の場合の電力: 9089 BTU/時
* 体重: 170.4ポンド (77.3 kg)
* 高さ: 4U
* 幅: 19インチ IEC ラック準拠 (17.6インチ 44.7 cm)
* 奥行き: 34.5インチ (ケーブル管理ブラケット付きで36.6インチ)
* 動作温度/高度/湿度: 標高3048m（10000フィート）まで10℃～35℃（50°F～95°F）、相対湿度8%～80%、結露なし
* 非動作時温度/湿度: -40°C ～ 70°C (-40°F ～ 158°F)、高度 12,192m (40,000 フィート) まで、相対湿度 10% ～ 95%、結露なし、元の容器内
* 音響騒音：表示音響出力（LwAd）：8.6 音圧（LpAm）（傍観者位置）：71.6 dB

* 安全性認証: BIS、CB、CSA、G_K_U-SoR、IRAM、NOM、NRCS、SONCAP、TBS
* 認証 安全性/EMC/EMI: EAC、UKRSEPRO
* 認証 安全性/EMC/EMI/RoHS: BSMI、CE DoC、UKCA DoC
* 規格 EMC/EMI: BS-EN-55032、BS-EN55035、CISPR 32、EN55022、EN55024、EN55032、EN55035、EN61000-3-2、EN61000-3-3、FCC Part 15 Class A、ICES-003、KS C 9832、KS C 9835
* 安全規格：ANSI/UL60950-1、ANSI/UL62368-1、BS-EN62368-1、CAN/CSA C22.2 No. 60950-1、CAN/CSA C22.2 No. 62368-1、CNS 15598-1、EN60825-1、EN62368-1、IEC 62368-1、IEC60950-1、IS 13252（パート1）
