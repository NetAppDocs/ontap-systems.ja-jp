= AFF C80の主な仕様
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
以下は、単一の高可用性ペアにおけるAFF C80 ストレージ システムの選択仕様です。このストレージ システムの完全な仕様については、 NetApp Hardware Universe (HWU) をご覧ください。



== AFF C80の主な仕様

* プラットフォーム構成: AFF C80 シングルシャーシ HA ペア
* 最大生容量: 14.7360 PB
* メモリ: 512.0000 GB
* フォームファクター: 2 つの HA コントローラと 48 個のドライブ スロットを備えた 4U シャーシ
* ONTAPバージョン: ONTAP: 9.16.1P2
* PCIe拡張スロット: 18
* 最小ONTAPバージョン: ONTAP 9.16.1RC1 == スケールアウトの最大値
* タイプ: NAS; HA ペア: 12; 物理容量: 176.8 PB / 157.0 PiB; 最大メモリ: 6144 GB
* タイプ: SAN; HA ペア: 6; 物理容量: 88.4 PB / 78.5 PiB; 最大メモリ: 3072 GB
* タイプ: HA ペア; 物理容量: 14.7 PB / 13.1 PiB; 最大メモリ: 512.0000
