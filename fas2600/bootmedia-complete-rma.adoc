= 障害のあるパーツを NetApp-FAS2600 に返却します
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
キットに同梱されている RMA 手順に従って、故障した部品をNetAppに返送してください。参照 https://mysupport.netapp.com/site/info/rma["パーツの返品と交換"] 詳細については、ページをご覧ください。FAS2600 システムは、手動のブート メディア回復手順のみをサポートします。自動ブート メディア リカバリはサポートされていません。
