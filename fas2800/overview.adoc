= FAS2820の主な仕様
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
以下は、単一の高可用性ペアにおけるFAS2820ストレージ システムの選択仕様です。このストレージ システムの完全な仕様については、 NetApp Hardware Universe (HWU) をご覧ください。



== FAS2820の主な仕様

* プラットフォーム構成: FAS2820シングルシャーシ HA ペア
* 最大生容量: 3.1680 PB
* メモリ: 128.0000 GB
* フォームファクター: 2Uシャーシ、2つのHAコントローラと12のドライブスロット
* ONTAPバージョン: ONTAP: 9.16.1P2
* PCIe拡張スロット: 4
* 最小ONTAPバージョン: ONTAP 9.13.1RC1




== スケールアウトの最大値

* タイプ: NAS; HA ペア: 12; 物理容量: 38.0 PB / 33.8 PiB; 最大メモリ: 1536 GB
* タイプ: SAN; HA ペア: 6; 物理容量: 19.0 PB / 16.9 PiB; 最大メモリ: 768 GB
* タイプ: HA ペア; 物理容量: 3.2 PB / 2.8 PiB; 最大メモリ: 128.0000

== システム環境仕様

* 標準電力: 1815 BTU/時
* 最悪の場合の電力: 2339 BTU/時
* 重量: 57.2ポンド (25.9 kg)
* 高さ: 2U
* 幅: 19インチ IEC ラック準拠 (17.6インチ 44.7 cm)
* 奥行き: 20.0インチ (ケーブル管理ブラケット付きで25.1インチ)
* 動作温度/高度/湿度: 標高3048m（10000フィート）まで10°C～35°C（50°F～95°F）、相対湿度8%～80%、結露なし
* 非動作時温度/湿度: -40°C ～ 70°C (-40°F ～ 158°F)、高度 12,192m (40,000 フィート) まで、相対湿度 10% ～ 95%、結露なし、元の容器内
* 音響騒音：表示音響出力（LwAd）：7.8 音圧（LpAm）（傍観者位置）：68.4 dB
