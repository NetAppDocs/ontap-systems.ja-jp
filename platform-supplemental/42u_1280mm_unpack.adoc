= システムキャビネットを開梱します
:icons: font
:imagesdir: ../media/


[role="lead"]
システムキャビネットを所定の位置に移動する前に、システムキャビネットを包む梱包材を取り除く必要があります。また、キャビネットを開梱したあとの梱包材はリサイクルする必要があります。
