= 使用する手順
:icons: font
:imagesdir: ../media/


[role="lead"]
プラットフォームの障害が発生したドライブを無停止で交換するには、プラットフォームがサポートするドライブに適した手順を選択します。SSD のホットスワップ用手順は回転式でないドライブ用、 HDD をホットスワップするための手順は回転式ドライブ用です。
